## HTTPS
https 协议是通过加入 SSL/TLS 来加密 HTTP 数据进行安全传输的 HTTP 协议，同时默认启用443端口进行数据传输。

## HTTPS 的加密过程
1. 客户端发起 HTTPS 请求
    - 用户在浏览器里输入一个网址，然后连接到服务器的443端口。

2. 服务端的配置
    - 采用 HTTPS 协议的服务器必须要有一套数字证书，可以自己制作，也可以向组织申请。
